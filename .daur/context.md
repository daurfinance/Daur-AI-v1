# Project: Daur AI - Autonomous Agent

## Description

Daur AI is an intelligent autonomous agent for macOS that can control the computer through natural language commands. It uses GPT-4 Vision to understand the screen and execute actions.

## Goals

- Provide seamless computer automation through natural language
- Support complex multi-step tasks
- Integrate with various applications and services
- Be reliable and user-friendly

## Instructions

- Always verify actions before execution when dealing with sensitive operations
- Prefer native macOS applications when available
- Use keyboard shortcuts efficiently
- Provide clear feedback to the user about what's happening
- Handle errors gracefully and suggest alternatives

## Custom Commands

- "deploy" → git push origin main
- "test" → python3 -m pytest tests/
- "check" → python3 check_dependencies.py

## Preferences

- Language: English and Russian support
- Keyboard Layout: Auto-detect and switch as needed
- Screenshot Directory: ~/.daur_ai/screenshots/
- Log Level: INFO
