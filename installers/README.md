# Установочные файлы Daur-AI v2.0

В этой директории находятся установочные файлы для Daur-AI v2.0 для различных операционных систем.

## Содержимое

### Установщики
- **Daur-AI-v2.0-Setup-Windows-x64.exe** - Установщик для Windows 10/11 (64-bit)
- **Daur-AI-v2.0-macOS-universal.dmg** - Установщик для macOS (Intel и Apple Silicon)

### Документация
- **WINDOWS_INSTALL.md** - Инструкция по установке для Windows
- **MACOS_INSTALL.md** - Инструкция по установке для macOS

### Скрипты
- **build-installers.sh** - Скрипт для сборки установщиков

## Системные требования

### Windows
- Windows 10/11 (64-bit)
- 8 ГБ ОЗУ (рекомендуется 16 ГБ)
- 4 ГБ свободного места на диске
- Процессор Intel Core i5 / AMD Ryzen 5 или выше
- Видеокарта с поддержкой OpenGL 3.3 или выше

### macOS
- macOS 11.0 (Big Sur) или новее
- Процессор Intel или Apple Silicon (M1/M2/M3)
- 8 ГБ ОЗУ (рекомендуется 16 ГБ)
- 4 ГБ свободного места на диске
