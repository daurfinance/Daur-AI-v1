===================================================
           Daur-AI v2.0 для macOS
===================================================

Спасибо за загрузку Daur-AI v2.0!

Для установки Daur-AI v2.0 на ваш компьютер:

1. Запустите файл install.sh, дважды щелкнув по нему
   (Если файл не запускается, откройте Terminal и выполните команду:
    chmod +x install.sh && ./install.sh)
2. Следуйте инструкциям на экране
3. После завершения установки запустите Daur-AI из Launchpad или папки Applications

Системные требования:
- macOS 11.0 (Big Sur) или новее
- Процессор Intel или Apple Silicon (M1/M2/M3)
- 8 ГБ ОЗУ (рекомендуется 16 ГБ)
- 4 ГБ свободного места на диске

Для получения дополнительной информации см. документацию в папке docs.
