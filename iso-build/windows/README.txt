===================================================
           Daur-AI v2.0 для Windows
===================================================

Спасибо за загрузку Daur-AI v2.0!

Для установки Daur-AI v2.0 на ваш компьютер:

1. Запустите файл install.bat, дважды щелкнув по нему
2. Следуйте инструкциям на экране
3. После завершения установки запустите Daur-AI с рабочего стола или из меню "Пуск"

Системные требования:
- Windows 10/11 (64-bit)
- 8 ГБ ОЗУ (рекомендуется 16 ГБ)
- 4 ГБ свободного места на диске
- Процессор Intel Core i5 / AMD Ryzen 5 или выше
- Видеокарта с поддержкой OpenGL 3.3 или выше

Для получения дополнительной информации см. документацию в папке docs.
