# Daur-AI v2.0 - Инструкции по тестированию на MacBook

## Обзор

Этот архив содержит исходный код и необходимые файлы для тестирования Daur-AI v2.0 на вашем MacBook. Проект включает в себя:

- Ядро искусственного интеллекта с встроенной языковой моделью
- Модули компьютерного зрения
- Интеграцию с Telegram
- Веб-интерфейс и десктопное приложение
- Автономное планирование задач

## Системные требования

- macOS 11.0 (Big Sur) или новее
- Python 3.9+ (рекомендуется Python 3.11)
- 8 ГБ ОЗУ (рекомендуется 16 ГБ)
- 4 ГБ свободного места на диске

## Установка

### 1. Установка Python (если не установлен)

Если у вас еще не установлен Python, скачайте и установите его с официального сайта:
https://www.python.org/downloads/macos/

### 2. Создание виртуального окружения

Откройте Terminal и выполните следующие команды:

```bash
# Перейдите в директорию проекта
cd путь/к/распакованному/архиву

# Создайте виртуальное окружение
python3 -m venv venv

# Активируйте виртуальное окружение
source venv/bin/activate
```

### 3. Установка зависимостей

```bash
# Установите необходимые пакеты
pip install -r requirements.txt
```

### 4. Настройка конфигурации

```bash
# Создайте файл конфигурации из шаблона
cp src/config/settings.py.example src/config/settings.py

# Отредактируйте файл конфигурации в любом текстовом редакторе
# Например:
nano src/config/settings.py
```

### 5. Загрузка встроенной языковой модели

```bash
# Запустите скрипт для загрузки модели
python scripts/download_embedded_model.py
```

## Запуск

### Запуск основного приложения

```bash
# Убедитесь, что виртуальное окружение активировано
source venv/bin/activate

# Запустите основное приложение
python src/main.py
```

### Запуск веб-интерфейса

```bash
# Запустите веб-сервер
python src/web/api_server.py
```

После запуска веб-интерфейс будет доступен по адресу: http://localhost:8000

### Запуск Telegram-бота

```bash
# Настройте токен бота в файле конфигурации
# Затем запустите бота
python src/telegram/daur_ai_bot.py
```

## Структура проекта

- `src/` - Исходный код проекта
  - `ai/` - Модули искусственного интеллекта
  - `browser/` - Модули для работы с браузером
  - `config/` - Конфигурационные файлы
  - `drivers/` - Драйверы для работы с устройствами
  - `executor/` - Исполнители команд
  - `input/` - Обработка ввода
  - `integration/` - Интеграции с внешними сервисами
  - `learning/` - Модули обучения
  - `logger/` - Логирование
  - `parser/` - Парсеры команд
  - `platforms/` - Платформо-зависимый код
  - `system/` - Системные модули
  - `telegram/` - Интеграция с Telegram
  - `ui/` - Пользовательский интерфейс
  - `vision/` - Модули компьютерного зрения
  - `web/` - Веб-интерфейс
  - `main.py` - Точка входа в приложение

## Тестирование

### Запуск тестов

```bash
# Запуск всех тестов
python -m unittest discover tests

# Запуск конкретного теста
python -m unittest tests/test_ai_model.py
```

### Тестирование компьютерного зрения

```bash
# Запуск демонстрации компьютерного зрения
python src/vision/demo.py
```

## Отладка

Для отладки рекомендуется использовать встроенные инструменты Python:

```bash
# Запуск с отладочной информацией
python -m pdb src/main.py
```
